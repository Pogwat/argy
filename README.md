# argy
A lua arguyment parsser designed to be as simple and minaimal as possible.

## Features

### Main
- [x] flags (cmd -f)
- [x] Arguments (cmd --arg -a)
- [x] Positional Arguments (cmd arg)
- [x] Help Generation

### Sub
- [x] Unused Arguments
- [ ] Arg types
- [ ] Custom Argument patterns

## Use

### Methods

Define your arg_types
Now only positional_arg, arg and flag are supported types:

Generic methods like the arg constructors are all generated by function generators so they have the same funciton input fields and similar names

These construct arg's fields to parse the command line args 

```lua
argy.inputs.args:arg(name,arg_ident, input_type, description)
argy.inputs.positional_args:positional_arg(name,arg_ident, input_type, description)
argy.inputs.flags:flag(name,arg_ident, input_type, description)
```

This parses the arg's and pushes them to argy.outputs.final_args.args

```lua
argy:gen_fargs() 
```

This generates the help handler (if you use the help.lua)

```lua
argy:gen_help() 
```

Now that gen_fargs created the argument entries in final args you can do 

```lua 
argy.outputs.final_args:get(name).value
```

To get the evaluated arg

### Example
```lua
-- Imports
require "argy"
require "help" --For help generation

-- Arg --hi as "hi"
argy.inputs.args:arg("hi","--hi", "string", "are you hi or are you bye")

-- Evalute args and push them to argy.outputs.final_args.args
argy:gen_fargs() 

-- This must be used after gen_fargs as it uses the value of help in final_args
-- gen_help() is optional you can just use core argy if you dont need help generation
argy:gen_help()

--Prints value of "hi" arg  
print(argy.outputs.final_args:get("hi").value)

```

## Installation

### Using luarocks
```bash
luarocks install --server=https://luarocks.org/dev argy
```

### Using Nix

Get the argy.nix file in this repo

```bash
git clone https://github.com/Pogwat/argy.git   
cd argy
cp argy.nix /etc/nixos #move it to the locaiton of your nix configs/flake
```


then add it to your overlays.
put this in configuraiton.nix:

```nix
nixpkgs.overlays = [

(final: prev: {
  argy = final.callPackage ./argy.nix {};
  #Rest of overlays...
}) 
];
```
